# Portfolio Website

Developer portfolio showcasing 12 full-stack projects with editable profile, work experience, and cover letter.

## Features
- Editable profile (name, email, GitHub, LinkedIn, bio)
- Projects grid linking to all 12 live apps
- Work experience timeline (add/edit/delete)
- Cover letter section
- All data stored in PostgreSQL

## Stack
React 19 · TypeScript · Tailwind CSS · Vite · React Query · Shadcn/UI

## Run
```bash
npm install && npm run dev
```
Requires [portfolio-api-server](https://github.com/WebGhost01/portfolio-api-server).
